# Sales Forecasting and Exploratory Data Analysis Project

## InternBoot Internship Project

### Author
Anthony Akpan

## Project Overview
This project focuses on Sales Forecasting and Exploratory Data Analysis (EDA) using Python and Machine Learning techniques.

The project was completed during the InternBoot Internship Program using the Kaggle Store Sales Time Series Forecasting dataset.

## Objectives
- Analyze sales trends and patterns
- Clean and preprocess sales data
- Perform exploratory data analysis
- Visualize sales performance
- Build forecasting models
- Predict future sales using regression techniques
